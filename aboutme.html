About me

My name is Victor, I'm from Ukraine. I like web design and I want to learn how to code. I am currently learning the following programming languages:

  • HTML - My first language, this one just makes the basic structure of a website.
  • CSS - This language gives color and beauty to the website.
  • Js - This language makes it possible to interact with the websites.
  • PHP - This is a widely used open source general purpose programming language. PHP is specially designed for web development and its code can be embedded directly into HTML.
  • SQL - This is a data-base language. It puts, removes, changes and finds Information on a very efficient way.
  • C - In CS50 they used this language to explain on a practical way how computers and memory work.
  • Python - With this language you can do nearly everything, like in C but in a much easier way.
